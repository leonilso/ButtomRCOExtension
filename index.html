Política de Privacidade da Extensão "Button RCO Extension"

Última atualização: 03 de novembro de 2025

1. Introdução

Bem-vindo à Button RCO Extension (doravante "Extensão"). Esta extensão foi desenvolvida por Leonilso (doravante "nós", "nosso"). Sua privacidade é extremamente importante para nós.

Esta política de privacidade explica quais informações coletamos (spoiler: nenhuma) e como as utilizamos.

2. Coleta e Uso de Dados

A Button RCO Extension é uma extensão de navegador simples projetada para operar inteiramente no seu computador local (client-side).

  • Não coletamos, armazenamos, usamos ou compartilhamos absolutamente nenhuma informação pessoal identificável.
  • Não coletamos dados de telemetria, análise ou de uso.
  • A extensão pode ler o conteúdo da página do RCO (Registro de Classe Online) exclusivamente para executar sua função principal (como copiar dados ou adicionar um botão). Essas informações são processadas localmente no seu navegador e nunca são enviadas para nossos servidores ou para terceiros.

3. Armazenamento de Dados

A extensão não armazena nenhuma informação pessoal. Se a extensão oferecer configurações personalizáveis, essas configurações serão salvas localmente no armazenamento do seu navegador (usando a API `chrome.storage` ou similar) e não serão acessíveis por nós.

4. Permissões da Extensão

A extensão solicita apenas as permissões mínimas necessárias para funcionar. Elas são usadas exclusivamente para permitir que o script da extensão seja executado nas páginas designadas do RCO e execute suas funções.

5. Alterações nesta Política

Podemos atualizar nossa Política de Privacidade de tempos em tempos. Quaisquer alterações serão publicadas nesta página. Recomendamos que você revise esta página periodicamente para se manter informado.

6. Contato

Se você tiver alguma dúvida sobre esta Política de Privacidade, pode entrar em contato através do repositório oficial no GitHub: github.com/leonilso/ButtonRCOExtension
